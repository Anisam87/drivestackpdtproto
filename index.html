Pipeline Architecture

RAW_DATA_STREAM
  ↓
[PROTOCOL_CLASSIFIER]
  ├─ OEM Match (Tata, Hyundai, MG, Kia, BYD, Mahindra)
  ├─ Standard Match (AIS-140, MQTT, CAN, Custom)
  └─ Fallback: Generic Parser
  ↓
[SCHEMA_MAPPER]
  ├─ Field Mapping (src → unified)
  ├─ Type Coercion
  ├─ Unit Conversion (km/h → m/s, etc.)
  └─ Null Handling
  ↓
[VALIDATION_LAYER]
  ├─ Range Checks
  ├─ Sanity Tests
  ├─ Anomaly Detection
  └─ Error Logging
  ↓
[ENRICHMENT]
  ├─ Geospatial Lookup
  ├─ Weather Context
  ├─ Road Conditions
  └─ Traffic State
  ↓
UNIFIED_OUTPUT (PostgreSQL + Redis)

Sample Transformation: Tata → Unified

INPUT (Tata_IoT_v2.1):
{
  "battery": {
    "soc": 67,
    "voltage": 398.4,
    "temperature": 28
  }
}

MAPPING RULES:
- battery.soc → vehicle.energy.state_of_charge
- battery.voltage → vehicle.energy.voltage_v
- battery.temperature → vehicle.energy.temp_celsius
- infer battery.capacity_kwh from historical SoC curves
- calculate vehicle.energy.estimated_range_km

OUTPUT (Unified Schema):
{
  "vehicle": {
    "energy": {
      "state_of_charge": 67,
      "voltage_v": 398.4,
      "temp_celsius": 28,
      "capacity_kwh": 50.5,
      "estimated_range_km": 337,
      "charging_status": "idle"
    }
  }
}

Transformation Rules (YAML)

--- protocols: - name: "Tata_IoT_v2.1" fields: - source: "telemetry.battery.soc" target: "vehicle.energy.state_of_charge" type: "integer" validation: "range(0, 100)" - source: "telemetry.location.latitude" target: "vehicle.position.latitude" type: "float" validation: "range(-90, 90)" - source: "telemetry.drivetrain.speed" target: "vehicle.motion.speed_kmh" type: "float" unit_conversion: "kmh" - name: "AIS-140" fields: - source: "vehicle_data.fuel_level" target: "vehicle.fuel.level_percent" type: "integer" validation: "range(0, 100)" - source: "location.speed" target: "vehicle.motion.speed_kmh" type: "float" unit_conversion: "kmh" enrichment: - source: "vehicle.position" enrich_with: ["weather", "traffic", "road_conditions"] timeout_ms: 500 fallback: "skip_enrichment" error_handling: - invalid_field: "log_and_skip" - missing_required: "infer_or_fail" - out_of_range: "clamp_or_flag"

Data Schema

Complete unified vehicle data model (340 fields)

Core Vehicle Schema (TypeScript)

interface Vehicle { // Identity id: string; vin: string; oem: string; model: string; year: number; registration_number?: string; // Position & Motion position: { latitude: number; longitude: number; altitude_m?: number; accuracy_m?: number; heading_degrees?: number; timestamp: ISO8601String; }; motion: { speed_kmh: number; acceleration_ms2?: number; rpm?: number; gear?: string; }; // Energy Management energy: { state_of_charge: number; // 0-100 voltage_v: number; current_a: number; temp_celsius: number; estimated_range_km: number; charging_status: 'idle' | 'charging' | 'error'; capacity_kwh: number; charge_port_status?: 'open' | 'closed' | 'error'; }; // Vehicle Status vehicle_status: { ignition: boolean; odometer_km: number; doors_locked: boolean; windows_open: boolean; sunroof_open?: boolean; trunk_open?: boolean; }; // Climate Control climate: { cabin_temp_celsius: number; setpoint_celsius: number; compressor_active: boolean; mode: 'cool' | 'heat' | 'auto' | 'off'; }; // Diagnostics diagnostics: { battery_health_percent: number; tire_pressure_fl: number; tire_pressure_fr: number; tire_pressure_rl: number; tire_pressure_rr: number; next_service_km: number; error_codes: string[]; fault_count: number; }; // Enrichment (Optional) enrichment?: { weather: WeatherData; traffic: TrafficData; road_conditions: RoadData; }; }

Extended Fields & Capabilities

// Safety & Security vehicle.safety { airbag_status: boolean; seatbelt_status: { driver: boolean; passenger: boolean; rear_left: boolean; rear_right: boolean; }; crash_detection: boolean; collision_count: number; } // Powertrain Details vehicle.powertrain { engine_type: 'ICE' | 'EV' | 'HYBRID' | 'PHEV'; fuel_type: 'petrol' | 'diesel' | 'electric' | 'hybrid'; transmission: 'manual' | 'automatic' | 'cvt' | 'direct'; drivetrain: 'FWD' | 'RWD' | 'AWD'; fuel_level?: number; fuel_consumption_lpl?: number; } // Connected Services vehicle.connected_services { remote_start: boolean; lock_unlock: boolean; climate_control: boolean; navigation_sync: boolean; ota_capable: boolean; last_sync_timestamp: ISO8601String; } // Insurance & Risk vehicle.risk { driver_behavior_score: number; accident_risk_score: number; theft_risk_score: number; maintenance_score: number; } // Fleet Management vehicle.fleet { fleet_id?: string; driver_id?: string; utilization_percent?: number; maintenance_status?: string; fuel_card_linked?: boolean; } // Battery & EV Specific vehicle.battery { pack_voltage: number; pack_capacity_kwh: number; usable_capacity_kwh: number; cell_count: number; chemistry: string; // 'LFP', 'NCA', 'NCM', etc. cycle_count: number; degradation_percent: number; cell_temp_min: number; cell_temp_max: number; balancing_active: boolean; }

Validation Rules & Constraints

VALIDATION SCHEMA: Field: vehicle.position.latitude Type: float Range: [-90, 90] Precision: 6 decimals Required: true Freshness: < 60 seconds Field: vehicle.energy.state_of_charge Type: integer Range: [0, 100] Unit: percentage Required: true (for EV) Freshness: < 5 seconds Field: vehicle.motion.speed_kmh Type: float Range: [0, 350] Precision: 1 decimal Required: true Freshness: < 2 seconds Field: vehicle.diagnostics.error_codes Type: array Length: [0, 100] Pattern: /^[A-Z0-9]{3,5}$/ Required: false Freshness: < 30 seconds ERROR HANDLING: - Out of Range: LOG_WARN + CLAMP_TO_LIMIT - Missing Required: LOG_ERROR + RETURN_NULL - Invalid Type: LOG_ERROR + INFER_TYPE - Stale Data (> Freshness): FLAG_STALE + INCLUDE
