🔄 Pipeline Stages
1 INGEST
Protocol Adapters
REST → JSON parser
MQTT → Topic subscriber
TCP → Binary frame decoder
XML → DOM parser
Auth Handler
OAuth 2.0 token refresh, client cert rotation, API key management per OEM
→
2 CLEAN
Type Coercion
String→Float: "18.5204" → 18.5204
String→Bool: "yes" → true
CSV→Array: "P0301,P0420" → ["P0301","P0420"]
Null normalize: null/""/[] → []
Outlier Detection
Speed > 250 km/h → flagged
Battery SoC > 100% → capped
GPS jumps > 10km/s → rejected
→
3 NORMALIZE
Unit Conversion
Knots → km/h (×1.852)
MPH → km/h (×1.609)
Miles → km (×1.609)
SoC 0-1 → 0-100%
Schema Mapping
All timestamps → ISO 8601 UTC
All locations → {lat, lng} floats
All speeds → km/h float
All DTCs → string array
→
4 ENRICH
